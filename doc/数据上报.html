基本信息

Path： /qyyx/log/record

Method： POST

接口描述：

请求参数

Headers

参数名称 参数值 是否必须 示例 备注
Content-Type application/x-www-form-urlencoded 是
token 是 传token

Query

参数名称 是否必须 示例 备注
upRequestId 是 上次请求id
requestId 是 请求id
sessionId 是 用户会话sessionId
source 是 用户设备源：小程序：wXapplet;安卓：android；苹果：ios
deviceId 是 设备唯一标识码，安卓使用androidId.苹果和小程序不需要
idfv 是 苹果专用idfv
phoneBrand 是 手机品牌；苹果传：iphone;
phoneModel 是 手机型号；苹果传eg：iPhone14,5
os 是 系统
net 是 网络格式:wifi、4G、5G
ua 是 用户user_agent
reasonSheng 否 地域省
reasonShi 否 地域市
reasonQu 否 地域区
type 是 openApp 埋点类型：app打开记录：openApp；用户点击记录：userClick；视频观看记录：videoPlayRecord；视频曝光：videoExposure；游戏曝光：gameExposure；转发记录：transpond；在线时长记录：onlineTime;搜索:search
data 是 app打开记录，openApp：{timeTemp(事件时间戳):1234234213,inform(其他信息预留字段，没有传空json):{}}，
用户点击记录，userClick：{
timeTemp(事件时间戳):1234234213
pageTime(秒):1205
item(页面编号)：LY0002
list:{
[timeTemp(事件时间戳):1234234213,item(页面编号)：LY0002,event(事件编号):LE00010,inform:{}],
[timeTemp(事件时间戳):1234234213,item(页面编号)：LY0002,event(事件编号):LE00010,inform:{}]
}
}

视频观看记录，videoPlayRecord：{timeTemp(事件时间戳):1234234213,videoId:123,duration(观看时长),inform(其他信息预留字段，没有传空json):{}}

视频曝光，videoExposure：{
list:{
[timeTemp(事件时间戳):1234234213,videoId:123,inform(其他信息预留字段，没有传空json):{}],
[timeTemp(事件时间戳):1234234213,videoId:123,inform(其他信息预留字段，没有传空json):{}]
}
}

游戏曝光,gameExposure：{
list:{
[timeTemp(事件时间戳):1234234213,gameId:123,inform(其他信息预留字段，没有传空json):{}],
[timeTemp(事件时间戳):1234234213,gameId:123,inform(其他信息预留字段，没有传空json):{}]
}
}

转发记录,transpond:{timeTemp(事件时间戳):1234234213,channel(分享渠道:qq,qqzone,weixin,weixinzone,app,weibo):qq,type(视频:video,游戏:game,APP:app,合辑:coauchor):game,soleId(对应的id):234,inform(其他信息预留字段，没有传空json):{}}


搜索or筛选后的点击,search:{timeTemp(事件时间戳):1234234213,type(视频筛选videoScreen,视频搜索videoSearch,综合搜索allSearch,用户搜索：userSearch,游戏搜索：gameSearch),clientType(video视频点击,user用户点击,game游戏点击):video,soleId（对应的id）:123,keyword:王者,seat(位置):15,inform(其他信息预留字段，没有传空json):{}}

在线时长记录,戳,onlineTime:{timeTemp(事件时间戳):1234234213,time(秒):156,inform(其他信息预留字段，没有传空json):{}}
